# RvR_blend

Blends two colors. Blend weights: 0.6*c0 + 0.4*c1. Swap c0 and c1 if you want to apply the opposite weight.

## Definition

```c
uint8_t RvR_blend(uint8_t c0, uint8_t c1);
```
