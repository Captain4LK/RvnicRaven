# RvR_fix24_divj

Divides two 24.8 fixed point numbers and returns the result

## Definition

```c
RvR_fix24 RvR_fix24_div(RvR_fix24 a, RvR_fix24 b);
```

## Notes

This functions doesn't have any protection against overflow or divide by zero.
