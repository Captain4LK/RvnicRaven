# RvR_fix16_sin

Calculates the sine of the angle a.

## Definition

```c
RvR_fix16 RvR_fix16_sin(RvR_fix16 a);
```

## Notes

Angles are given as a value between 0 and 1 (0 and 65536), so

* 32768 = pi rad = 180°
* 65536 = 2 pi rad = 360°
