# RvR_fix16_div

Dividies two 16.16 fixed point numbers and returns the result

## Definition

```c
RvR_fix16 RvR_fix16_div(RvR_fix16 a, RvR_fix16 b);
```

## Notes

This functions doesn't have any protection against overflows or division by zero.
