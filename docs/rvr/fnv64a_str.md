# RvR_fnv64a_str

Function for calculating the hash of a string using the fnv64a hash. Additionally you can specify the initial value of the hash (hval), this can be used to chain hashes.

## Definition

```c
uint64_t RvR_fnv32a_str(const char *str, uint64_t hval);
```
